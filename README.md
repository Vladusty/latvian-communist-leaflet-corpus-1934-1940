# Latvian Communist Leaflet Corpus (LCLC)

This repository contains the **Latvian Communist Leaflet Corpus (LCLC)** and related research materials.

## Repository structure

- `LCLC_1934-1940/` — official dataset (CLARIN-LV release, version 1.0)
- `workspace/` — development files, scripts, intermediate data, and analysis (not part of the dataset)

## Dataset

The curated and citable version of the corpus is located in the `LCLC_1934-1940` directory.

This version corresponds to the dataset published in CLARIN-LV (v1.0).

The GitHub repository may contain updated or extended versions of the data.

## Description

The Latvian Communist Leaflet Corpus (1934–1940) is a structured digital corpus of underground political leaflets produced by illegal communist organizations in Latvia during the interwar period.

For full documentation, see:
→ `LCLC_1934-1940/README.md`
